Interactive Graphics Quadrants

A web implementation of my Java graphics program showing various rendering techniques:

  • Random shapes generation
  • Color manipulation
  • 3D visualization techniques

Quadrants Description

  • Top-Left: Random colored lines
  • Top-Right: Colored squares of fixed size
  • Bottom-Left: Random colored circles of varying sizes
  • Bottom-Right: 3D cube

The original version of this program was written in Java for my CS-182 Intro to Java Programming course at Cuyamaca College, using the Eclipse IDE. The initial code was provided by my professor, Mr. David.

We received the initial code, which included the background, white borders, and partial content for the Upper-Left Quadrant. Our task was to complete the Upper-Left Quadrant and also develop the Bottom-Left and Top-Right Quadrants. The Bottom-Right Quadrant, featuring a 3D Cube, was available for extra credit.

Below, I have included the .java file that I submitted for my coursework. This serves as a reference for the original code. I later ran the code through Replit to create an interactive version, which is displayed above and features controls created with JavaScript.
