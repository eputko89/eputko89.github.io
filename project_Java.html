Interactive Graphics Quadrants

A web implementation of my Java graphics program showing various rendering techniques:

  • Random shapes generation
  • Color manipulation
  • 3D visualization techniques

Quadrants Description

  • Top-Left: Random colored lines
  • Bottom-Left: Random colored circles of varying sizes
  • Top-Right: Colored squares of fixed size
  • Bottom-Right: 3D cube

The original version of this program was written in Java for my CS-182 Intro to Java Programming course at Cuyamaca College using the Eclipse IDE.

Initial code was provide by my Professor, Mr. David, from Cuyamaca College.

We were provided with the initial code of the background, white borders, and partial content of the Upper-Left Quadrant. We were instructed to finish the Bottom-Left and Top-Right quadrants. The Bottom-Right quadrant was available for extra credit.

I've provided my submitted school work java src file below to reference as the original code before running it through Replit to help me display an interactive version on my page which was made using JavaScript and is displayed above.
